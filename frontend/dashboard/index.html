🧠 AgencyOS Architecture

3D Visual Engine: Everything you see in the map is rendered natively in WebGL using Three.js. As Python emits AI decision events, GSAP translates those into walk/type animations.

Multi-Agent Workflow: When you start a campaign, a local Python Orchestrator manages the flow. The Strategist queries the Local RAG Engine for market trends, then assigns tasks to specialists like the Content Writer or SEO Agent.

Governance & Approvals: Tasks don't just finish. The Risk Agent and QA Approver intercept the raw text generated by the Local LLMs (TinyLlama/Qwen). If they detect brand risks, they reject the task and send it back for Revision.

Data Persistency: Every token generated, copy written, and dollar logged is stored in a SQLite database and retrievable via the Company Drive.
